# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-26

### Fixed
- Fixed known bugs

### Added
- AI configuration options

### Removed
- Removed existing Ollama from the project

## [0.4.1] - 2025-11-18

### Added
- Auto-start on system boot with corresponding settings
- Quick search can now be activated with Option+Space
- Ability to search for applications and files
- AI Mark label for files that have been AI-marked
- New search algorithm for more accurate search results

## [0.4.0] - 2025-11-11

### Added
- Keyboard shortcut: Press Alt+Space to quickly access the search bar and find files more easily
- System tray: Closing the application now minimizes to the system tray in the bottom-right corner instead of completely closing
- Enhanced search categories: You can now search for folders and applications in the search bar activated by the keyboard shortcut

## [0.3.1] - 2025-10-31

### Changed
- Multi-language support updates
- Automatic update check functionality

## [0.3.0] - 2025-10-27

### Added
- OCR text recognition functionality
- AI marking feature
- Right-click context menu functionality
- OCR indexing failure handling
- Automatic AI model download
- CUDA package handling optimization

## [0.2.3] - 2025-10-15

### Changed
- Service startup optimization
- AI image summary optimization

### Fixed
- AI image summary error fixes

### Added
- Desktop shortcut creation

## [0.2.2] - 2025-10-11

### Changed
- GPU compatibility refactoring
